# Cantina Bella Roma - Landing Page

Esta é a estrutura estática portável da landing page da Cantina Bella Roma.

## Estrutura de Arquivos
- `index.html` - Estrutura principal da página
- `css/style.css` - Todos os estilos da aplicação (CSS customizados, regras responsivas e animações de keyframes)
- `js/script.js` - Toda a lógica e animações da aplicação (Interatividade, carrosséis, formulários, lightbox e animações GSAP)
- `images/` - Imagens da aplicação
- `icons/` - Ícones da aplicação
- `fonts/` - Fontes locais da aplicação
